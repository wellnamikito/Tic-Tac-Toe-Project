# 🎮 Tic-Tac-Toe (JavaFX)

## 📌 Описание

Tic-Tac-Toe — настольное приложение, реализующее классическую игру «Крестики-нолики» с графическим интерфейсом на JavaFX.

Проект демонстрирует работу с GUI, игровой логикой и базовой архитектурой Java-приложения.

---

## 🚀 Возможности

- 🎮 Игра между двумя игроками на одном устройстве
- 🤖 Игра против компьютера 
- 🧠 Проверка корректности ходов
- 🏆 Автоматическое определение победителя или ничьей
- 🖥️ Графический интерфейс на JavaFX
- ⚙️ Настройки приложения

---
