# 🎮 Tic-Tac-Toe

## 📌 Описание

Tic-Tac-Toe — это настольное приложение, реализующее классическую игру «Крестики-нолики» с графическим пользовательским интерфейсом.

Программа поддерживает несколько режимов игры и предоставляет пользователю удобный способ взаимодействия с игровым процессом.

---

## 🚀 Возможности

- 🎮 Игра между двумя игроками на одном устройстве  
- 🤖 Игра против компьютера  
- 🌐 Сетевая игра (клиент-сервер)  
- 🧠 Проверка корректности ходов  
- 🏆 Автоматическое определение победителя или ничьи  
- 🖥️ Графический пользовательский интерфейс  
- ⚙️ Настройки игры  
- 💾 Сохранение параметров и (при необходимости) статистики

## ▶️ Запуск

1. Установите приложение  
2. Запустите файл `Game.exe`  
3. Выберите режим игры в главном меню  

---

## 🛠️ Технологии

- Java  
- Swing / JavaFX  
- TCP-сокеты (для сетевого режима)  

---

## 📂 Структура проекта

- `Game` — игровая логика  
- `UI` — пользовательский интерфейс  
- `Network` — сетевой модуль  
- `Config` — настройки приложения
